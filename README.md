A full-stack AI chatbot web application built using **JavaScript, Node.js, Express, MongoDB**, and a modern **ChatGPT-style frontend UI**.  
This project provides a real-time conversational interface where users can chat with an AI assistant, and all conversations are stored in MongoDB.

The UI is inspired by ChatGPT:
- Full-screen dark theme  
- Wide chat area  
- User messages on the right as bubbles  
- Bot responses as clean text blocks  
- Pill-shaped input box with circular send button  
- `Enter` to send, `Shift + Enter` for new line
